# OpenForlorn

OpenForlorn is an open-source project built with **cocos2d-x v2.2.3**. This repository targets the legacy cocos2d-x engine and is separate from the Axmol version.

## Building

### Requirements

- Visual Studio 2013 Community Edition
- cocos2d-x v2.2.3

### Steps

1. Install Visual Studio 2013 Community Edition.
2. Download and set up cocos2d-x v2.2.3.
3. Clone this repository.
4. Open the solution in Visual Studio and build the project.
